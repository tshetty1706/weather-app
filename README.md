# 🌦️ Weather App (Frontend)

A responsive weather web application built using **HTML, CSS, and JavaScript**.
This project displays real-time weather data with dynamic UI updates and background videos based on weather conditions.

---
---

## 🙌 Note

This is a **practice project** built to strengthen frontend development skills and understand API integration.

---

## 🚀 Features

* 🔍 Search weather by city name
* 🌡️ Displays temperature, humidity, wind speed, pressure
* 🎥 Dynamic background videos based on weather (Sunny, Rain, Snow, Clouds)
* 🌙 Day/Night detection using sunrise & sunset data
* 🕒 Live date and time
* ⏳ Loading indicator for better UX
* 📱 Fully responsive design

---

## 🛠️ Tech Stack

* HTML5
* CSS
* JavaScript
* OpenWeather API

---

## ⚙️ How It Works

* Fetches weather data using OpenWeather API
* Updates UI dynamically based on API response
* Maps weather conditions to corresponding background videos
* Uses sunrise & sunset timestamps to detect day/night

## 🎥 Video Assets

Background videos used in this project are not included in the repository to keep it lightweight and optimized.

You can download similar free videos from:

* https://www.pexels.com/videos/
* https://pixabay.com/videos/
* https://coverr.co/

Suggested search keywords:

* "sunny sky timelapse"
* "rain falling slow motion"
* "snow falling"
* "cloud moving sky"

After downloading, place videos inside a `videos/` folder:

weather-app/
│── videos/
│   ├── sunny.mp4
│   ├── rain.mp4
│   ├── snow.mp4
│   ├── cloudy.mp4


## ⚡ Setup Instructions

1. Clone the repository
2. Open `index.html` in your browser
3. Add your own API key in `weather.js`
